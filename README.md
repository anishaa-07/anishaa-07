## 🟣 Featured Projects

<details open>
<summary><b>🚀 Huddle — Collaborative Team Workspace</b></summary>
<br/>

A self-conceived, full-stack collaborative workspace merging the best of **GitHub, Notion, and Discord** — built for teams to manage projects, track tasks, and communicate in real time, all in one place.

| Category | Details |
|---|---|
| **Stack** | Node.js, Express, MongoDB/Mongoose, JWT, Socket.io, Cloudinary, React, Tailwind CSS |
| **Scale** | Multi-team, multi-project architecture with role-based membership |
| **Performance** | Real-time messaging via Socket.io with efficient room-based broadcasting |
| **Security** | JWT-based authentication with protected route middleware |
| **Impact** | Consolidates team management, task tracking, and communication into a single tool |
| **Repository** | [github.com/anishaa-07/huddle](https://github.com/anishaa-07) |

Huddle is architected around core entities — **User, Team, TeamMember, Project, Task, Message,** and **FileAttachment** — supporting Kanban-style task boards, per-project real-time chat, file uploads, and a live progress dashboard. Designed and built from scratch, from schema design through API architecture to real-time infrastructure.

</details>
